# Data Structures in C

A collection of standalone C implementations of classic data structures
and algorithms. Each source file is self-contained.

## Structure

All sources live under `src/`:

- basic_linked_list.c
- circular_linked_list.c
- doubly_linked_list.c
- doubly_circular_linked_list.c
- stack_linked_list.c
- stack_array.c
- queue_array.c
- queue_linked_list.c
- binary_search_tree.c
- sorts.c
- division_hash.c
- double_hash.c
- chaining_int.c

## Build

Compile any file with a C compiler. Example:

gcc src/basic_linked_list.c -o basic_linked_list

## Notes

These are learning examples focused on clarity and core operations.
